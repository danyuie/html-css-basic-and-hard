# **CSS IN HTML ?**

## **1. Internal**

sử dụng 1 cặp thẻ `<style>` trong file html.

## **2. External**

liên kết file css vào html và viết code css trong file css.

## **3. Inline**

sử dụng thuộc tính `style ` trong thẻ.
